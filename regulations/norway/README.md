# Regulations — Norway

Norwegian petroleum legislation, PSA regulations, and NPD guidelines relevant to upstream joint-venture governance.

## Primary Legislation

| Document | Year | Notes |
|----------|------|-------|
| Petroleum Act (`petroleumsloven`) | 1996 (last amended) | Core licensing and operatorship framework |
| Petroleum Regulations | 1997 | Detailed rules under the Act |
| CO₂ Tax Act (offshore) | 1991 | Environmental cost internalisation |
| NPD Data Regulations | 2020 | Mandatory data submission to NPD |
| PSA Framework Regulations | 2010 | Safety and working environment |

## CCS/CCUS Sources — Northern Lights JV

Primary sources on the Northern Lights CO₂ transport and storage JV (Equinor / Shell / TotalEnergies, 33.3% each), the operational vehicle for Norway's *Longship* full-scale CCS programme. Gassnova represents the Norwegian state as funding and oversight counterpart.

|--------|-------------|
| [Northern Lights — Reports portal](https://norlights.com/reports-2/) | Official publications hub: annual reports, carbon footprint studies, monitoring assessments |
| [Concept Report RE-PM673-00001 (2020)](https://ccsnorway.com/wp-content/uploads/sites/6/2020/05/Northern-Lights-Project-Concept-report.pdf) | Pre-FID JV formation document; Gassnova study agreement; open-source T&S infrastructure design |
| [Carbon Footprint — T&S Value Chain (2023)](https://norlights.com/wp-content/uploads/2023/11/Report-Carbon-footprint-of-the-Northern-Lights-JV-co2-transport-and-storage-value-chain.pdf) | Lifecycle GHG assessment: 97.4% net CO₂ abatement over 25-year asset life |
| [Annual Benefits Realisation Report to Gassnova (2025)](https://gassnova.no/app/uploads/sites/6/2025/04/Annual-Benefits-Realisation-Report-Northern-Lights-to-Gassnova.pdf) | State-oversight accountability report; Gassnova/Ministry of Energy performance monitoring mechanism |
| [Annual Experience Transfer Report 2023 — Phase 1 Facilities](https://gassnova.no/app/uploads/sites/6/2024/04/PM673-JV-A-PG-00015_02-Annual-experience-transfer-report-2023-for-Northern-Lights-Phase-1-Facilities.pdf) | Contractual knowledge-sharing to Gassnova; operational lessons (quay design, pipeline cleanliness, CCS-specific specifications) |
